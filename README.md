# All-In-One_Decoder
This is a decoder / converter I made for CTF challenges. It takes in one input and outputs a bunch of functions at once so you can find the answer quickly.

Currently Does:

base64

base32

hex to decimal

hex to binary

binary to hex

binary to decimal

decimal to binary

decimal to hex

decimal to Ascii

Ascii to decimal

Octal to decimal

Morse code to Ascii

Rail Fence Cipher

ROT13 (1-26)

At-bash Ciphe
